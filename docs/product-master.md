# 商品マスター

Rinker 登録商品と、記事との対応を管理する台帳です。

**方針：** 運営者による実購入・使用レビューは行わない。選定根拠は Amazon / 楽天 / 公式の **評価・口コミ・スペック**（執筆時点で再確認）。

---

## 選定フィルター（共通）

| 条件 | 目安 |
| --- | --- |
| 評価 | ★4.0 以上（★3.8 以下は原則除外） |
| レビュー数 | Amazon 100件以上 / 楽天 50件以上 |
| ASP | **Rinker 登録可能**（Amazon API 不可の場合は楽天/Yahoo で第三者店舗に出ている商品を優先） |
| 価格帯 | 消耗品 1,000〜5,000円（記事ペルソナに合わせて調整） |
| 在庫 | 欠品・終売でない |
| ターゲット | 「20代専用」等、45歳ペルソナと明確にズレるものは除外 |

記事あたり：**主役1本 ＋ 比較0〜2本**。主役の乱立はしない。
